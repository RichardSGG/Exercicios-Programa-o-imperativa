# Exercícios de Programação Imperativa

Repositório de resoluções em Java para a disciplina de Programação Imperativa na UNICAP.

## Conteúdo
- **Lista1ponto**: Resolução de 20 questões fundamentais (Q1 a Q20).
- **Lista0**: Exercícios complementares da disciplina.
- **Documentação**: Enunciados em PDF e notas de apoio para referência.

## Estrutura
Os exercícios estão organizados em pastas por listas, com cada questão num ficheiro individual para facilitar a consulta e o estudo.

## Como Executar
Para compilar e executar uma questão através do terminal (exemplo Q6):

```powershell
javac -d . ProgImperativa\Lista1ponto\Q6.java
java Lista1ponto.Q6
